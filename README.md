# DesignPatternsApp

## 📌 Жоба туралы

Бұл жоба Java тілінде жазылған және екі негізгі мінез-құлықтық (Behavioral) паттернді көрсетеді:

- Strategy (Стратегия)
- Observer (Наблюдатель)

Жобаның мақсаты — жобалау шаблондарын практикалық түрде көрсету және олардың қолданылу логикасын түсіндіру.

---

# 🔹 1. Strategy (Стратегия) паттерны

## 📖 Анықтама

Strategy паттерны — алгоритмдерді жеке кластарға бөліп, оларды бір-бірінен тәуелсіз қолдануға мүмкіндік беретін шаблон.

Яғни, біз алгоритмді (біздің жағдайда — төлем тәсілін) бағдарлама құрылымын өзгертпей ауыстыра аламыз.

---

## 🎯 Жобадағы қолданылуы

Біз төлем жүйесін модельдедік.

Қолданушы келесі төлем әдістерінің бірін таңдай алады:

1. Банк картасы
2. PayPal
3. Криптовалюта

Әр төлем тәсілі жеке класс ретінде жүзеге асырылған.
